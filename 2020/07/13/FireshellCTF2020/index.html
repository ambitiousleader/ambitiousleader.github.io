https://web.archive.org/web/20191220171022/https://www.darkmatter.ae/blogs/breaching-the-perimeter-phantomjs-arbitrary-file-read/

这篇文章对于这道题的洞讲解的很详细，就是通过file://加载的文件被认为是符合Same Origin Policy的同源文件

URL TO PDF

访问url会形成pdf文件并且下载

用requestbin来看一下请求

发现使用了WeasyPrint，查一下资料有关WeasyPrint的说明

  • Only fetched images
  • No JavaScript
  • No<iframe>
  • we can use:<img> <embed> <object> <link>

根据Ying师傅的wp，问题在<link>标签，在标签内href加载file://`实现SSRF+任意文件读取

访问，将下载来的pdf文件放入kali里，kali自带了pdfdetach工具
